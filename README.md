# Proxygen
Proxygen is a simple proxy management wrapper around the Linode client.

# Installation

Ubuntu
------

Clone the repository
```bash
git@github.com:vanctran/proxygen.git
```

Give executable permissions and begin the installation
```bash
chmod u+x setup.sh
sudo ./setup.sh
```
